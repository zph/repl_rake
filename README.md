## Installation

Add this line to your application's Gemfile:

    gem 'repl_rake'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install repl_rake

## Usage

Then in gem folder add this to Rakefile

    require 'repl_rake'
    ReplRake.setup
